# Portfolio Archive

Portfolio Archive は、自分の制作物を管理するためのデスクトップアプリです。
React + Vite で画面を作成し、Tauri を使って Windows 向けデスクトップアプリとして動作するようにしています。

## 使用技術

* React
* Vite
* Tauri
* JavaScript
* CSS
* JSON

## 主な機能

* 制作物の登録
* 制作物の編集
* 制作物の削除
* ステータス管理
* キーワード検索
* ステータス絞り込み
* 制作物数の集計表示
* 関連ファイルの添付
* 添付ファイル名と保存パスの表示
* 添付ファイルの削除
* 添付ファイルの保存場所を開く

## データ保存

登録した制作物データは、Tauri のファイルシステム機能を使って `projects.json` に保存しています。
ブラウザの `localStorage` には依存せず、アプリ専用のデータ保存領域に JSON ファイルとして保存する構成です。

保存データはアプリ本体とは別の領域に保存されるため、アプリをアンインストールしても、ユーザーデータを削除しない設定であれば保存データを残すことができます。

## ファイル添付について

関連ファイルは、ファイル本体をアプリ内にコピーするのではなく、元ファイルのパスを記録する方式です。
そのため、元ファイルを移動または削除すると、記録されたパスからは参照できなくなります。

## 開発環境での起動方法

依存パッケージをインストールします。

```powershell
npm install
```

開発環境でアプリを起動します。

```powershell
npm run tauri dev
```

## 配布用ビルド

Windows向けの配布用ファイルを作成する場合は、以下のコマンドを実行します。

```powershell
npm run tauri build
```

ビルド後、以下のようなファイルが生成されます。

直接起動用の実行ファイル：

```text
src-tauri\target\release\portfolio-archive.exe
```

MSIインストーラー：

```text
src-tauri\target\release\bundle\msi\Portfolio Archive_0.1.0_x64_en-US.msi
```

NSISセットアップ：

```text
src-tauri\target\release\bundle\nsis\Portfolio Archive_0.1.0_x64-setup.exe
```

## インストール確認

NSISセットアップを使用して、以下の動作を確認済みです。

* インストールできる
* スタートメニューに表示される
* デスクトップショートカットを作成できる
* アプリを起動できる
* 既存データを読み込める
* 新規データを保存できる
* 再起動後も保存データが保持される
* アンインストールできる
* ユーザーデータを削除しない場合、再インストール後も保存データが保持される
